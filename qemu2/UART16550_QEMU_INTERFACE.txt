Machine: vexpress-a9 / uartarm-soc
UART base address: 0xff210000
UART region size: 0x1000

Compatible UART model:
UART 16550A compatible (basic register model)

Board-compatible mapping:
0xff210000 = 0xff200000 lightweight HPS-to-FPGA bridge + 0x00010000 UART IP

Standard registers:
0x00 : RBR / THR / DLL
0x04 : IER / DLM
0x08 : IIR / FCR
0x0C : LCR
0x10 : MCR
0x14 : LSR
0x18 : MSR
0x1C : SCR

Extension registers:
0x20 : TX_COUNT
0x24 : RX_COUNT
0x28 : ERR_COUNT
0x2C : IRQ_COUNT

Current implemented behavior:
- DLAB switching through LCR[7]
- DLL / DLM access
- THR write
- RBR read
- IER read/write
- FCR write (basic reset behavior)
- LCR read/write
- MCR read/write
- LSR read
- MSR read
- SCR read/write
- TX_COUNT / RX_COUNT / ERR_COUNT / IRQ_COUNT read/write

Current limitations:
- FIFO behavior is simplified
- Interrupt generation is simplified
- Timing is not modeled
- RX path is test-driven, not from external serial line
